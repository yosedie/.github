# yosedie/.github

This is the default configuration and community health repository for the **[@yosedie](https://github.com/yosedie)** GitHub account.

## Purpose

The files in this repository provide fallback defaults for all other public repositories under my account that do not define their own specific templates or guidelines. 

## Included Files

- **[SECURITY.md](SECURITY.md)**: Security vulnerability reporting instructions.
- **[CONTRIBUTING.md](CONTRIBUTING.md)**: Guidelines on how to contribute to my projects.
- **[CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)**: Standard contributor code of conduct to ensure a welcoming environment.
- **[SUPPORT.md](SUPPORT.md)**: Standard help and support channels.
- **[PULL_REQUEST_TEMPLATE.md](PULL_REQUEST_TEMPLATE.md)**: Pull Request checklist template.
- **[ISSUE_TEMPLATE/](.github/ISSUE_TEMPLATE/)**: Default Issue templates for Bug Reports and Feature Requests.
